# repro-stackoverflow-keys

Repro for SO ssl keys example.
https://stackoverflow.com/questions/52116642/how-can-i-use-rustls-to-encrypt-a-byte-slice-using-a-known-public-key-and-pkcs1

## Usage
```sh
$ cargo run --example main
```
